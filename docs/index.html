「現在行なっている研究」

使用言語：C#
環境：Unity,VScode
VRを用いた聴覚に着目した視覚障害の体験についての研究を行っています。
視覚に障害のある人は見える人に比べ、聞くことに注意を向けています。また、人によっては失った視力を埋め合わせるように聴力が発達することがあります。
れまでの視覚障害を体験するVRでは見え方に着目したものでしたが、本研究では聞こえ方に着目した体験を研究することでよりリアルな体験を実現することを目指しています。
